# 🤖 Mentor de Carreira com IA

## 🎯 Objetivo

Este projeto simula um mentor de carreira em tecnologia utilizando inteligência artificial para orientar usuários na escolha de áreas de atuação.

O sistema conduz uma entrevista estruturada e sugere carreiras personalizadas com base no perfil do usuário.

---

## ⚙️ Como Funciona

O mentor realiza 7 perguntas para entender:

- Interesses (dados, código ou pessoas)
- Experiência prévia
- Disponibilidade de estudo
- Objetivo profissional
- Preferências de trabalho
- Interesses em tecnologia
- Experiência anterior relevante

Com base nas respostas, ele sugere 3 carreiras mais adequadas.

---

## 🧩 Estrutura da Solução

1. Coleta de informações do usuário  
2. Análise do perfil  
3. Recomendação de carreiras  
4. Justificativa personalizada  

---

## 📊 Exemplo de Resultado

Carreiras sugeridas:

- Analista de Dados  
- Cientista de Dados  
- Engenheiro de Dados  

Com base em:
- Interesse em dados  
- Conhecimento em estatística  
- Interesse em inteligência artificial  

---

## 💡 Insights

- A personalização melhora a assertividade das recomendações  
- A base em estatística é um diferencial em carreiras de dados  
- O modelo permite adaptação para diferentes perfis
