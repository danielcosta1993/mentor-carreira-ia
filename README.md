
# 🤖 Mentor de Carreira com IA

Mentor de carreira com Inteligência Artificial que analisa o perfil do usuário e sugere caminhos em tecnologia de forma personalizada.

---

## 🎯 Objetivo

Este projeto simula um mentor de carreira em tecnologia utilizando IA para orientar usuários na escolha de áreas de atuação.

O sistema realiza uma entrevista estruturada, analisa o perfil do usuário e sugere carreiras e planos de estudo personalizados.

---

## 🎯 Público-Alvo

- Pessoas em transição de carreira  
- Iniciantes em tecnologia  
- Profissionais indecisos sobre qual área seguir  

---

## 🔄 Fluxo do Sistema

1. Usuário responde 7 perguntas  
2. IA analisa o perfil  
3. Sugere 3 carreiras mais compatíveis  
4. Gera um plano de estudos personalizado  

---

## ⚙️ Como Funciona

O mentor realiza perguntas para entender:

- Interesses (dados, código ou pessoas)  
- Experiência prévia  
- Disponibilidade de estudo  
- Objetivo profissional  
- Preferências de trabalho  
- Interesses em tecnologia  
- Experiência anterior relevante  

Com base nessas informações, a IA recomenda as melhores carreiras.

---

## 🧠 Estrutura da Solução

- Coleta de informações do usuário  
- Análise do perfil  
- Recomendação de carreiras  
- Justificativa personalizada  
- Geração de plano de estudos  

---

## 📊 Exemplo de Resultado

### Carreiras sugeridas:

- 🥇 Cientista de Dados  
- 🥈 Analista de Dados  
- 🥉 Engenheiro de Dados  

### Base da recomendação:

- Interesse em dados  
- Conhecimento em estatística  
- Interesse em inteligência artificial  

---

## 💡 Insights

- A personalização melhora a assertividade das recomendações  
- A base em estatística é um diferencial em carreiras de dados  
- IA pode apoiar decisões de carreira de forma estruturada
